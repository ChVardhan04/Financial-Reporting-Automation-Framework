# Financial Reporting Automation Framework

## Overview
This project automates financial reporting by extracting financial data,
transforming it using Python and PySpark, storing it in BigQuery, and
visualizing insights using a Streamlit dashboard.

## Architecture
Data Source (Kaggle / Yahoo Finance)
        ↓
Python & PySpark ETL
        ↓
BigQuery Data Warehouse
        ↓
Streamlit Dashboard

## Technologies
Python
PySpark
BigQuery
SQL
Streamlit
Google Cloud

## How to Run
1. Install dependencies
2. Configure BigQuery credentials
3. Run Streamlit dashboard

streamlit run app.py
